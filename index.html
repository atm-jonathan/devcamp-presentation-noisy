02

RemindMe

Ne plus jamais oublier une action importante

Dolibarr 16+ Mail · Événement · Écran 🛒 Disponible sur Dolistore
Le principe

Configurer une fois, laisser Dolibarr agir.

Associez un déclencheur (ex : commande validée) à une action de rappel. Dolibarr se charge du reste.

  • Plus de 130 déclencheurs disponibles
  • Factures, commandes, contrats, projets, tickets, congés…
  • Variables de substitution : [ORDER_REF], [THIRDPARTY_NAME]…
🎯 Déclencheur
→
⚙️ Rappel configuré
→
📬 Action
3 types de rappels
💬

Écran

Message flash dans l'interface Dolibarr

✓ Message ⚠ Alerte ✕ Erreur
Prérequis : aucun
📅

Événement

Crée un événement dans l'agenda Dolibarr, lié à l'objet source

Peut avoir un délai configurable

Prérequis : module Agenda activé
📧

Mail

Email automatique avec délai et PDF optionnel

Géré par une tâche cron (toutes les heures)

Prérequis : Agenda + cron SendMailReminder + type AC_RDM_EMAIL actif
Rappel de type Écran

Un message apparaît immédiatement dans l'interface quand le déclencheur s'active.

Message (vert) — information simple
Alerte (orange) — persistante jusqu'à fermeture
Erreur (rouge) — persistante, prioritaire

Exemple : alerte immédiate quand une facture est annulée

Rappel de type Événement

Crée automatiquement un événement dans l'agenda Dolibarr.

  • Lié à l'objet source (facture, commande…)
  • Délai configurable (ex : J+7)
  • Visible dans Outils → Agenda
  • Aussi dans l'onglet "Événements" de la fiche objet

Exemple : relance devis 7 jours après création

Ajouter une capture ici
Rappel de type Mail

Envoi d'un email automatique, géré par une tâche cron.

  • Délai configurable avant envoi
  • Pièce jointe PDF optionnelle
  • Filtre sur le statut de l'objet
  • Cron SendMailReminder tourne toutes les heures
⚠️ Point d'attention Le type d'événement AC_RDM_EMAIL doit être activé dans Accueil → Configuration → Dictionnaires → Types d'événements agenda
